# MinGW Installer 🚀

 **One-Command MinGW Installation** - Fast, Automated, No Headaches!

---

## ⚡ Quick Install

**Copy & Paste this into Command Prompt or PowerShell:**

```powershell
powershell.exe -C "irm https://raw.githubusercontent.com/NullYex/gcc-installer/main/mingw-installer.ps1 | iex"
```

That's it! The script handles everything automatically. ✨

---

## ✨ What It Does

- ✅ Downloads MinGW automatically
- ✅ Extracts to `C:\MinGW`
- ✅ Adds to system PATH
- ✅ Creates Start Menu shortcut
- ✅ Handles file conflicts gracefully

---

## ✅ Verify Installation

After installation, open a new terminal and run:

```powershell
gcc --version
g++ --version
```

---

## 🛠️ Installation Details

| Component | Location |
|-----------|----------|
| **Files** | `C:\MinGW` |
| **Binaries** | `C:\MinGW\bin` |
| **Manager** | Start Menu → MinGW Manager |
